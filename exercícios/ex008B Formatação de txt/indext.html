Segundo Jeff Noble, no seu livro HTML para leigos:

A diferença entre elementos de bloco e elementos em linha é que os elementos de bloco ocupam toda a largura disponível, enquanto os elementos em linha ocupam apenas o espaço necessário para o seu conteúdo. Por exemplo, um parágrafo é um elemento de bloco, enquanto uma palavra dentro de um parágrafo é um elemento em linha.
